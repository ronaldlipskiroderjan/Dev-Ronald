📊 SalesAnalytics API

Finalizado ✅

O SalesAnalytics automatiza a geração de relatórios de vendas em Excel direto do Banco de dados.

O foco técnico é a implementação de uma API REST em Spring boot integrada ao PostgreSQL via Spring Data JPA, com ênfase na manipulação de arquivos binários (Apache POI) para automação de relatórios corporativos.

Destaques Técnicos:
  • ☕ Java e Spring Boot
  • 🍃 Spring Data JPA
  • 🐘 PostgreSQL
  • 📊 Apache POI
